# 04.5 - Operacion Frontend

## Arranque recomendado

El frontend se ejecuta en Docker Compose junto con backend.

```bash
docker compose up --build
```

## Comandos utiles de calidad

En local (solo para validacion/IDE):

```bash
cd frontend
corepack enable
corepack prepare yarn@4.12.0 --activate
yarn install
yarn lint
yarn build
```

## Variables de entorno frontend

Ver `frontend/.env.example`.

- `VITE_API_URL`
- `VITE_API_BASE_URL`
- `VITE_SOCKET_PATH`
- `VITE_GOOGLE_CLIENT_ID`
- `VITE_NEWS_FORCE_MOCKS`

## Checklist rapido de smoke test UI

1. Home carga completa con Header/Footer.
2. Login/registro funcionan visualmente.
3. `/noticias` lista noticias sin errores de contrato.
4. `/noticias/:slug` abre detalle.
5. Panel admin carga estructura base.
