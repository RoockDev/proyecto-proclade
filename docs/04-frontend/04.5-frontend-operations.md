# 04.5 - Operacion Frontend

## Arranque recomendado

En stack completo:

```bash
docker compose up --build
```

## Comandos utiles locales (IDE/QA)

```bash
cd frontend
corepack enable
corepack prepare yarn@4.12.0 --activate
yarn install
yarn lint
yarn build
yarn dev
```

## Variables de entorno frontend

Referencia: `frontend/.env.example`

- `VITE_API_URL`
- `VITE_API_BASE_URL`
- `VITE_SOCKET_PATH`
- `VITE_GOOGLE_CLIENT_ID`
- `VITE_NEWS_FORCE_MOCKS`

## Checklist rapido de smoke test UI

1. Header muestra estado correcto de sesion (sin "sesion fantasma" por token expirado).
2. `/noticias` y `/noticias/:slug` cargan sin errores de contrato.
3. `/superheroes` carga lazy + paginacion.
4. `/bibliotecas-humanas/:delegationSlug` resuelve tabs/contacto/mapa y libros.
5. Rutas `/admin/*` redirigen a login si no hay sesion valida.
6. Chatbot abre, responde, sugiere y permite feedback.
