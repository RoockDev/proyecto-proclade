# 04.3 - Integracion API y Contratos

## Cliente HTTP

- Archivo: `frontend/src/services/http/axios.instance.ts`
- Base URL:
  - `VITE_API_BASE_URL` si esta definido
  - fallback `http://localhost:3000`

## Contrato compartido

- Tipo: `frontend/src/types/api.ts`

```ts
export type ApiResponse<T> = {
  success: boolean;
  message: string;
  data: T | null;
};
```

## Auth API

Archivo: `frontend/src/features/auth/api/auth.api.ts`

Funciones principales:

- `login(payload)`
- `register(payload)`
- `googleSignIn(payload)`
- `forgotPassword(payload)`
- `resetPassword(payload)`

## News API

Archivo: `frontend/src/features/news/api/news.api.ts`

Funciones:

- `getLatestNews(limit)`
- `getNewsList()`
- `getNewsBySlug(slug)`

### Modo mock controlado por env

- Variable: `VITE_NEWS_FORCE_MOCKS`
- Si `true`, frontend usa mocks de `news.mocks.ts`.
- Si `false`, consume backend real.

## Mapeo API -> UI

- Archivo: `frontend/src/features/news/utils/news.mapper.ts`
- Responsabilidades:
  - Normalizar shape de noticia para UI.
  - Formatear fecha segura.
  - Ordenar por fecha de publicacion.
  - Construir estado de paginacion cliente.
