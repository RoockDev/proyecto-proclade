# 04.3 - Integracion API y Contratos

## Cliente HTTP

- Archivo: `frontend/src/services/http/axios.instance.ts`
- Base URL:
  - `VITE_API_BASE_URL` si existe
  - fallback `http://localhost:3000`

## Contrato compartido

- Tipo: `frontend/src/types/api.ts`

```ts
export type ApiResponse<T> = {
  success: boolean;
  message: string;
  data: T | null;
};
```

## Gestion de sesion

Archivos clave:

- `features/auth/utils/auth-session.storage.ts`
- `services/http/axios.instance.ts`

Comportamiento:

1. Se guarda `accessToken` + `authUser` en login/register/google.
2. `getAuthSession()` valida expiracion del JWT (`exp`) antes de devolver sesion.
3. Si el token expiro, limpia localStorage automaticamente.
4. Interceptor Axios limpia sesion cuando recibe `401`.

## Modo mock controlado

Noticias mantienen switch temporal:

- Variable: `VITE_NEWS_FORCE_MOCKS`
- `true`: usa mocks locales.
- `false`: consume backend real.
