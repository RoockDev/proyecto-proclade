# 04.1 - Visión frontend

## Stack

- React 19
- TypeScript
- Vite 7
- React Router 7
- Axios
- Bootstrap 5
- Socket.IO Client

## Organización por feature

Carpetas principales:

- `features/home`
- `features/news`
- `features/superheroes`
- `features/human-libraries`
- `features/challenges`
- `features/colabora`
- `features/auth`
- `features/profile`
- `features/chatbot`
- `features/privacy`
- `features/admin`
- `router`
- `services/http`

## Layouts principales

### `PublicLayout`

Incluye:

- `Header`
- `Outlet`
- `Footer`
- `ChatbotWidget`
- `CookiePreferencesModal`

### `AdminLayout`

Incluye:

- sidebar
- topbar
- guard de UX según sesión/rol

## Responsabilidades destacadas

- sesión en `localStorage`
- limpieza de sesión al detectar JWT expirado o `401`
- rutas públicas y admin en el mismo frontend
- consentimiento de servicios externos desde cliente
- tiempo real para el panel admin del chatbot

## Estado funcional actual

- no hay proveedor social de login
- no hay store global tipo Redux
- el estado se reparte entre hooks, APIs y componentes por feature
