# 04.1 - Vision Frontend

## Stack

- React 19
- TypeScript
- Vite 7
- React Router
- Axios
- Bootstrap 5 + CSS propio

## Estructura por features

Carpetas principales:

- `features/auth`: autenticacion y recovery.
- `features/home`: home publica y secciones de marca.
- `features/news`: listado/detalle de noticias y paginacion cliente.
- `features/admin`: panel administrativo y componentes compartidos admin.
- `router`: definicion de rutas/layouts.
- `services/http`: instancia Axios central.
- `styles`: estilos globales y overrides Bootstrap.

## Layouts

- Publico: `PublicLayout` con `Header`, `Outlet`, `Footer`.
- Admin: `AdminLayout` para zona interna.

## Principios de implementacion

1. Tipado estricto para contratos con backend.
2. Componentes reutilizables por feature.
3. Estado local por pagina/componente (sin store global compleja).
4. API client centralizado.
