- Mantener unit tests por servicio en reglas de negocio.

2. Frontend
- Introducir tests de componentes criticos (header auth state, admin guards, chatbot widget, formularios admin).

3. Integracion
- Smoke de rutas publicas y admin tras deploy staging.

## Definition of Done tecnica sugerida

Una HU se considera cerrada cuando cumple:

1. Contrato API respetado.
2. Lint y build verdes en componentes tocados.
3. Cobertura de camino feliz + error principal.
4. Documentacion tecnica actualizada en `docs/`.
