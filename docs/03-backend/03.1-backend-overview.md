# 03.1 - Vision Backend

## Stack

- NestJS 11
- TypeScript
- Prisma 7
- PostgreSQL
- JWT
- class-validator/class-transformer

## Modulos activos

1. `AuthModule`
- Login, registro, Google Sign-In, forgot/reset password.
- Emision de JWT.

2. `UsersModule`
- CRUD de usuarios (solo admin).
- Asignacion de roles.
- Soft delete de usuarios.

3. `NewsModule`
- Lectura publica de noticias publicadas.
- Gestion admin de noticias (CRUD con soft delete).
- Generacion de slugs unicos.

4. `PrismaModule`
- Servicio de acceso a datos.

5. `MailModule`
- Envio de correos de recuperacion de contrasena.

## Bootstrap y middleware global

Definidos en `backend/src/main.ts`:

- CORS configurable por `CORS_ORIGIN`.
- ValidationPipe global:
  - whitelist
  - forbidNonWhitelisted
  - transform
  - stopAtFirstError
- TransformInterceptor global para respuesta unificada.
- GlobalExceptionFilter para error unificado.

## Contrato de salida backend

```ts
{
  success: boolean;
  message: string;
  data: T | null;
}
```

Este contrato es obligatorio para frontend y debe mantenerse estable.
