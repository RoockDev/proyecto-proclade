## Convenciones vigentes

1. `deletedAt` aplica soft delete por defecto en varias entidades.
2. Los assets no viven en la base de datos, solo se guarda su URL local.
3. No existe un modelo específico para login social porque ya no hay Google auth.
4. La aceptación de privacidad en registro se persiste en `User.privacyAcceptedAt`.

## Bootstrap y datos base

El proyecto ya no depende de un seed general de usuarios demo.

En su lugar:

- `backend/prisma/bootstrap-system.ts`
  - asegura `ADMIN` y `USER`
  - crea o verifica el admin inicial
  - carga la base inicial del chatbot

## Ficheros y límites de uploads

- noticias:
  - `/uploads/news`
  - imágenes hasta 5 MB
- superhéroes:
  - `/uploads/superheroes`
  - imágenes hasta 2 MB
- libros humanos:
  - `/uploads/human-books`
  - PDF hasta 10 MB
