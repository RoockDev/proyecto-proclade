# 05.1 - Entorno local con Docker

## Servicios definidos

`docker-compose.yml` levanta:

1. `db` (PostgreSQL 16)
2. `redis` (Redis 7)
3. `backend` (NestJS)
4. `frontend` (React + Vite)
5. `nginx` (reverse proxy)

## Flujo de arranque

```bash
docker compose up --build
```

Validaciones basicas:

- App: `http://localhost`
- API via Nginx: `http://localhost/api/...`
- Assets backend: `http://localhost/uploads/...`

## Volumenes persistentes

- `db_data`
- `backend_node_modules`
- `frontend_node_modules`
- bind mount `./backend/uploads:/app/uploads`

## Reset completo (uso excepcional)

```bash
docker compose down -v
docker compose up --build
```

Usar solo cuando se necesite limpieza total de base de datos y estado local.
