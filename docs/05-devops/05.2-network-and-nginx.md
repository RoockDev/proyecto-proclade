# 05.2 - Topologia de red y Nginx

## Rol de Nginx

Archivo: `nginx/default.conf`.

Nginx escucha en `:80` y enruta:

- `/` -> `frontend:5173`
- `/api/` -> `backend:3000/`
- `/socket.io/` -> `backend:3000/socket.io/`

## Implicaciones

1. El frontend puede vivir sin exponer puerto publico directo.
2. El backend queda encapsulado detras de Nginx para API y sockets.
3. Cambios de rutas proxy afectan directamente a consumo frontend.

## Recomendaciones para produccion

1. Añadir TLS (443) y redireccion HTTP -> HTTPS.
2. Añadir cabeceras de seguridad (`HSTS`, `X-Frame-Options`, `CSP`).
3. Definir rate limits basicos para rutas sensibles (`/auth/*`).
