# 02.1 - C4 Context

## Objetivo

Mostrar actores y sistemas externos que interactuan con la plataforma.

```mermaid
flowchart LR
  guest["Visitante"]
  member["Usuario registrado"]
  admin["Administrador ONG"]

  platform["Plataforma Proclade<br/>Web fullstack para contenido y gestion"]

  google["Google Identity<br/>OAuth/OpenID"]
  smtp["SMTP Provider<br/>Envio correo reset"]

  guest -->|"HTTPS"| platform
  member -->|"HTTPS + JWT"| platform
  admin -->|"HTTPS + JWT"| platform

  platform -->|"Verifica idToken (HTTPS)"| google
  platform -->|"Envia reset password (SMTP)"| smtp
```

## Notas

- El sistema se expone en web publica y panel admin dentro del mismo frontend.
- El backend concentra reglas de negocio y seguridad.
- Integraciones externas actuales: Google Identity y SMTP.
