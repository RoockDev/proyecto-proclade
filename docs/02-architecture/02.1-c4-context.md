# 02.1 - C4 Context

## Objetivo

Mostrar qué actores y sistemas externos interactúan hoy con la plataforma real.

```mermaid
flowchart LR
  guest["Visitante público"]
  member["Usuario registrado"]
  admin["Administrador"]

  platform["Proyecto Proclade / Equipo PUCH<br/>web pública + panel admin + chatbot"]

  smtp["Proveedor SMTP"]
  maps["Google Maps<br/>servicio externo opcional"]
  legal["Web oficial Fundación PROCLADE<br/>aviso legal / privacidad"]

  guest -->|"HTTP"| platform
  member -->|"HTTP + JWT"| platform
  admin -->|"HTTP + JWT"| platform

  platform -->|"envío de email"| smtp
  platform -->|"iframe / enlace externo, solo con consentimiento"| maps
  platform -->|"enlaces legales externos"| legal
```

## Notas

- La autenticación actual es únicamente email/contraseña.
- El acceso admin comparte frontend con la web pública, pero el backend aplica guards específicos.
- Google Maps no se carga por defecto: depende de las preferencias de cookies/servicios externos del usuario.
- El chatbot forma parte del producto público y tiene una capa de administración separada.
