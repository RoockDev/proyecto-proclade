# 02.3 - C4 Component (Backend)

## Objetivo

Documentar componentes internos del contenedor backend (NestJS).

```mermaid
flowchart LR
  fe["Frontend React"]
  db[("PostgreSQL")]
  google["Google Identity"]
  smtp["SMTP"]

  subgraph api["NestJS API (componentes principales)"]
    authController["AuthController"]
    authService["AuthService"]
    usersController["UsersController"]
    usersService["UsersService"]
    newsController["NewsController"]
    adminNewsController["AdminNewsController"]
    newsService["NewsService"]
    jwtGuard["JwtAuthGuard"]
    rolesGuard["RolesGuard"]
    transformInterceptor["TransformInterceptor"]
    exceptionFilter["GlobalExceptionFilter"]
    prismaService["PrismaService"]
    googleAuthService["GoogleAuthService"]
    mailService["MailService"]
  end

  fe -->|"POST /auth/*"| authController
  fe -->|"CRUD /users (admin)"| usersController
  fe -->|"GET /news"| newsController
  fe -->|"CRUD /admin/news (admin)"| adminNewsController

  authController --> authService
  usersController --> usersService
  newsController --> newsService
  adminNewsController --> newsService

  authService --> usersService
  authService --> googleAuthService
  authService --> mailService

  authService --> prismaService
  usersService --> prismaService
  newsService --> prismaService

  prismaService --> db
  googleAuthService --> google
  mailService --> smtp

  authController --> jwtGuard
  usersController --> jwtGuard
  adminNewsController --> jwtGuard
  usersController --> rolesGuard
  adminNewsController --> rolesGuard

  authController --> transformInterceptor
  newsController --> transformInterceptor
  authController --> exceptionFilter
  newsController --> exceptionFilter
```

## Nota

Componentes compartidos (interceptor/filtro/guards) aplican transversalmente al backend.
