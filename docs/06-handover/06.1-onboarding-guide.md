# 06.1 - Guia de onboarding tecnico

## Objetivo

Permitir que un nuevo desarrollador levante y entienda el proyecto en menos de 1 jornada.

## Paso 1 - Clonar y preparar

```bash
git clone <repo-url>
cd proyecto-proclade
```

## Paso 2 - Configurar variables

1. Copiar `.env.example` -> `.env` en raiz.
2. Copiar `backend/.env.example` -> `backend/.env`.
3. Copiar `frontend/.env.example` -> `frontend/.env`.

## Paso 3 - Levantar entorno

```bash
docker compose up --build
```

## Paso 4 - Validar sistema

1. Abrir `http://localhost`.
2. Verificar `GET /api/news` responde.
3. Probar login basico con usuario de seed si aplica.

## Paso 5 - Comprension minima del codigo

1. Leer `docs/01-project-overview.md`.
2. Leer `docs/02-architecture/*`.
3. Revisar modulos backend en `backend/src/*`.
4. Revisar `frontend/src/features/*`.
