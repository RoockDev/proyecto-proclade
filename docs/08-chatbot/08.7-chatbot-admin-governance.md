# 08.7 - Admin chatbot, gobernanza y tiempo real

## Funciones del panel admin

El panel `/admin/chatbot` permite:

- ver métricas
- listar preguntas no resueltas
- resolver, reabrir o borrar unresolved
- crear y editar knowledge items
- crear y editar intents
- crear y editar phrases
- ajustar configuración del motor

## Tiempo real

### Backend

- gateway admin:
  - `backend/src/chatbot/admin/chatbot-admin.gateway.ts`
- gateway público que reenvía eventos:
  - `backend/src/chatbot/chatbot.gateway.ts`

### Frontend

- servicio cliente:
  - `frontend/src/services/http/admin-chatbot-websocket.service.ts`

Eventos relevantes:

- `adminConnected`
- `newMessage`
- `unresolvedQuestionCreated`

## Riesgo actual importante

El namespace `/admin/chatbot` no está protegido por guard de autenticación en el gateway. A efectos documentales debe considerarse una deuda prioritaria.

## Gobernanza operativa recomendada

1. revisar unresolved con frecuencia
2. promover preguntas frecuentes a knowledge items
3. ajustar pesos y umbrales con cambios pequeños y medibles
4. comprobar que las respuestas dinámicas siguen siendo coherentes cuando cambia contenido público
